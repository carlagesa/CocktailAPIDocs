# Lookup Endpoints

Lookup endpoints allow you to retrieve detailed information about specific cocktails and ingredients.

## Lookup cocktail by ID {{ tag: 'GET', label: '/v1/lookup/cocktail' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a cocktail by its unique ID.

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/lookup/cocktail">

    ```bash {{ title: 'cURL' }}
    curl -G https://api.protocol.chat/v1/lookup/cocktail \
      -H "Authorization: Bearer {token}" \
      -d id="cocktail_001"
    ```

    ```js
    import ApiClient from '@example/protocol-api'

    const client = new ApiClient(token)

    await client.cocktails.lookupById('cocktail_001')
    ```

    ```python
    from protocol_api import ApiClient

    client = ApiClient(token)

    client.cocktails.lookup_by_id(id="cocktail_001")
    ```

    ```php
    $client = new \Protocol\ApiClient($token);

    $client->cocktails->lookup_by_id('cocktail_001');
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "id": "cocktail_001",
      "name": "Mojito",
      "ingredients": ["Mint", "Lime", "Sugar", "Rum"],
      "instructions": "Muddle mint and lime, add sugar and rum, top with soda water.",
      "image_url": "https://example.com/images/mojito.jpg",
      "created_at": 692233200,
      "updated_at": 705103200
    }
    ```

  </Col>
</Row>
